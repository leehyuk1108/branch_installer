How To Use

Device flow

  1. Open a card and copy the direct installer URL.
  2. On the comma device, choose Custom Software.
  3. Paste the direct /installer URL.
  4. The device downloads an ELF installer, then that installer clones the target Git branch.

Important

GitHub Pages is static hosting. For arbitrary repos and live generation, run the optional server mode on your own machine.
