사람이 보기 쉬운 카탈로그와 실제 설치에 쓰이는 ELF 인스톨러 파일을 함께 제공합니다. comma 기기에는 이 페이지 주소가 아니라 짧은 별칭 또는 직접 인스톨러 URL을 넣어야 합니다.

예시: https://github.com/leehyuk1108/sunny-hl/tree/release-c3-hl 를 넣으면, 등록된 짧은 별칭이 있을 때 바로 보여줍니다.

사용 방법

기기 설치 흐름

  1. 카드를 열고 직접 인스톨러 URL을 복사합니다.
  2. comma 기기에서 Custom Software를 선택합니다.
  3. 직접 /installer URL을 붙여넣습니다.
  4. 기기가 ELF 인스톨러를 받은 뒤, 그 인스톨러가 대상 Git 브랜치를 clone 합니다.

주의

GitHub Pages는 정적 호스팅입니다. 임의 repo를 즉시 생성하려면 별도 서버 모드 또는 Netlify Functions를 사용해야 합니다.
